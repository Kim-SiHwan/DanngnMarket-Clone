# ShMarket
당근마켓과 같은 서비스를 제공하는 프로젝트
---
### 프로젝트 목표 
1. 견고한 어플리케이션 제작 
2. 성능을 개선하는 리팩토링 진행하며 기록과 검증
3. 메시지 큐를 사용한 웹 푸시 알람 제작, 채팅 기능 제작
4. Git을 통한 깔끔한 프로젝트 관리
5. 단순 CRUD만을 사용하는 것이 아닌 기술의 고도화, 성능 리팩토링
---
### 추가해야할 기능
- SMTP 메일 인증 ( 비밀번호 변경과 회원가입 )
- 회원 탈퇴, 비밀번호 변경 
- FCM 푸시 한번만 가는거 개선
- 게시글 수정
- 테스트
